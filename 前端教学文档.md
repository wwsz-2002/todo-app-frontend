# 现代待办事项应用前端教学文档

## 适用人群
本教程适用于前端初学者，尤其是对Java后端有一定了解但前端知识基础薄弱的开发者。

## 项目概述
我们将学习如何使用原生HTML5、CSS3和JavaScript创建一个功能完整、界面现代的待办事项(Todo)应用。

## 技术栈
- **HTML5**：页面结构
- **CSS3**：页面样式和响应式设计
- **JavaScript**：交互功能和数据处理
- **localStorage**：本地数据存储

## 项目结构
```
todo-app/
├── index.html          # 主HTML文件（包含内联JavaScript）
├── styles.css          # CSS样式文件
└── 前端教学文档.md      # 本教学文档
```

## 核心功能
1. 任务添加
2. 任务状态管理（完成/未完成）
3. 任务删除
4. 任务统计
5. 数据持久化

## 详细教程

### 第一部分：HTML结构

#### 1. 基本结构
HTML是网页的骨架，我们使用语义化标签来构建页面结构：

- `<!DOCTYPE html>`：声明文档类型
- `<html>`：根元素
- `<head>`：包含元信息和外部资源
- `<body>`：页面内容

#### 2. 页面布局
我们将页面分为三个主要部分：

- **头部**：显示应用标题
- **输入区域**：用于添加新任务
- **任务列表**：显示所有待办事项
- **统计区域**：显示任务完成情况

#### 3. 表单元素
- `<input type="text">`：文本输入框，用于输入任务内容
- `<button>`：按钮，用于提交任务
- `<input type="checkbox">`：复选框，用于标记任务完成状态

### 第二部分：CSS样式

#### 1. CSS变量
使用CSS变量管理颜色和样式参数，使代码更易维护：

```css
:root {
    --primary-color: #3498db;  /* 主色 */
    --secondary-color: #e74c3c;  /* 辅助色 */
    /* 其他变量... */
}
```

#### 2. 布局样式
- **容器**：限制最大宽度，居中显示
- **响应式设计**：使用媒体查询适配不同屏幕尺寸
- **Flexbox**：用于灵活的布局排列

#### 3. 视觉效果
- **圆角**：增加界面的现代感
- **阴影**：增强界面的深度感
- **过渡动画**：提升用户体验

#### 4. 响应式设计
- **桌面端**（≥1024px）：最大宽度600px
- **平板端**（768px-1023px）：最大宽度500px
- **手机端**（≤767px）：自适应屏幕宽度

## 常见问题解答

### 1. 什么是语义化标签？
语义化标签是指能够清晰表达其内容含义的HTML标签，如`<header>`、`<section>`、`<footer>`等。使用语义化标签有助于提高代码的可读性和可维护性，同时对搜索引擎优化也有帮助。

### 2. 什么是CSS变量？
CSS变量是一种存储值的方法，可以在整个样式表中重复使用。使用CSS变量可以使样式代码更易维护，当需要修改颜色、字体大小等参数时，只需要修改一处即可。

### 3. 什么是localStorage？
localStorage是浏览器提供的一种本地存储机制，可以在用户的浏览器中存储数据，即使页面刷新或关闭后重新打开，数据也不会丢失。localStorage存储的数据是以键值对的形式存在的，且只能存储字符串类型的数据。

### 4. 什么是DOM？
DOM（Document Object Model）是文档对象模型，是HTML和XML文档的编程接口。它表示文档的结构，允许程序动态地访问和修改文档的内容、结构和样式。

### 5. 什么是事件委托？
事件委托是一种事件处理模式，它利用事件冒泡机制，将事件监听器添加到父元素上，而不是每个子元素上。这样可以减少事件监听器的数量，提高性能，尤其是在处理大量子元素的情况下。
